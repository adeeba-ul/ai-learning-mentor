
# 🧭 CourseCompass
AI STRATEGIC COMPASS — "When you're lost, we hand you the map."

## 📋 About
- **CourseCompass** is an empathetic, high-fidelity AI Learning Mentor engineered for individuals navigating career confusion, choice paralysis, or personal growth plateauing.
- The platform breaks traditional academic silos by executing a **multi-dimensional diagnostic analysis** across a user’s current technical skills, psychological/personality blind spots, financial trajectories, and professional goals. 
- Using this computed multi-goal matrix, it curates a direct, tailored portal window into the live `coursera.org` ecosystem alongside non-academic strategic books and video frameworks to close hidden lifestyle and capability gaps.
- **Tech Stack:** React, TypeScript, Vite, Tailwind CSS, Supabase (Authentication, Managed Storage, Relational Database Realtime Engines), and server-side Gemini AI orchestration.

## 🧭 CourseCompass - Features 
The application provides a zero-friction, ultra-premium glassmorphism workspace optimized to transfer structural user context into clean, actionable milestones:

* **Strategic Home / Landing Engine:** Introduces the "Frictionless Five-Step Compass" methodology, complete with real-time vector previews spanning Mindset, Technical Core, Financial Planning, and Project Deployment.
* **Mentorship Command Dashboard:** Houses the global **Holistic Weakness Diagnostic Panel** which surfaces the top three operational threats to the user’s career trajectory, financial scaling, and personal focus.
* **4-Step Context Discovery Wizard:** Collects multi-dimensional parameters including current tech vectors, target affinity spaces, performance inhibitors, and strategic milestones.
* **Dynamic Certificate Parsing Dropzone:** An asset workflow engine utilizing backend OCR logic. Users drop image or PDF certificates into secure Supabase storage buckets to automatically append credential histories into active state tables.
* **Unified Profile Architecture:** Centralizes complex user parameters, multi-year income goals, and historic baseline inputs for persistent application references.
* **Curated Coursera Portal & Recommendation Engine:** A beautiful interactive card carousel tracking Affinity Match percentages, complete with an **AI Mentor Justification Block** detailing precisely *why* a particular curriculum path is mandatory for your specific life trajectory.
* **Chronological Learning Roadmap:** Generates a vertical glowing master timeline tracking multi-layered career milestones (e.g., Engineering Core -> Deep Learning Mastery -> MLOps Deployment -> Senior Specialization).
* **The Self-Improvement Growth Vault:** A custom media index tracking curated alternative assets (Strategic books like *Atomic Habits* or *The Psychology of Money* alongside curated technical/lifestyle deep dives) targeted to resolve foundational personal habits.
* **Saved Courses Buffer:** A direct utility space allowing users to bookmark, monitor, and remove selected Coursera path tracking blocks seamlessly.
